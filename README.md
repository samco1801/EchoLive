# EchoLive

EchoLive es una aplicación de escritorio gratuita y de código abierto desarrollada en Python que permite leer en voz alta los comentarios de un TikTok LIVE utilizando voces de inteligencia artificial.

## Características actuales

-  Conexión a TikTok LIVE
-  Detección automática cuando el LIVE está activo
-  Lectura de comentarios en tiempo real

##  Próximas funciones

-  Voz IA (Text-to-Speech)
-  Filtro anti-spam
-  Cola inteligente de comentarios
-  Interfaz gráfica
-  Integración con TikTok LIVE Studio
-  Sistema de configuración

##  Tecnologías

- Python 3
- TikTokLive
