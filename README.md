# EchoLive

EchoLive es una aplicación de escritorio gratuita y de código abierto desarrollada en Python que permite leer en voz alta los comentarios de un TikTok LIVE utilizando voces de inteligencia artificial.

---

## Características

- Conexión en tiempo real con TikTok LIVE.
- Lectura automática de comentarios mediante Edge TTS.
- Voces de inteligencia artificial.
- Estadísticas en tiempo real.
- Filtro inteligente de comentarios.
- Eliminación automática de emojis.
- Cola de comentarios para evitar interrupciones.
- Sistema de configuración.
- Interfaz gráfica desarrollada con CustomTkinter.

---

## Tecnologías

- Python 3
- TikTokLive
- edge-tts
- CustomTkinter
- Pillow
- asyncio
